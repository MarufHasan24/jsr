JSR

JSR stands for javascript reverse. It is a programming language that is designed for fun. It is not absulatlly a reverse version of javascript. It is a complicated language. It is compiled to javascript. You have to write this programme in a file with .jsr extension but from the bottom. That means the last line of your code will be the first line of your file. And the first line of your code will be the last line of your file.

Warning: Don't miss any semicolon. If you miss any semicolon, you will get an error. Warning: Don't use any keyword as a variable name. If you use any keyword as a variable name, you will get an error. Warning: Must include std to start your code. If you don't include std, you will get an error.#include<std>

Problem is not end here. The blocks of this code will run from top to bottom, like other programmes. But the lines of a block will run from bottom to top. That means the last line of a block will run first and the first line of a block will run last.

Confused?

Let's see an example. Suppose you want to print "Hello, world!" in the console. In javascript, you will write this code: 

console.log("Hello, world!");
But in jsr, you have to write this code: 

print("Hello, world!");
#include<std>
This was prity easy, right!

But now, let's see a complex example. Suppose you want to print the sum of two numbers. In javascript, you will write this code: 


    let a = 10;
    let b = 20;
    let c = 0;
    console.log(c); // 0
    function sum(a, b) {
        c += a + b;
        return c;
    }
    console.log(sum(a, b));// 30
But in jsr, you have to write this code: 

print(c); /* 0 */
velt c = 0;
velt b = 20;
velt a = 10;
print(c); /*Error : Cannot access 'c' before initialization*/
#include
function sum1(a, b) {
    return c;
    c += a;
    c += a + b;
}
function sum2(a, b) {
    c += a + b;
    return c;
}
print(sum1(a, b)); /* 40 */
print(sum2(a, b)); /* undefined */
This was a little bit complex, right!

You can use any text editor to write this code. But you have to compile this code here.

Variables

In jsr, you can declare variables using the following keywords:

  • velt Used to declare a mutable variable.
  • velc Used to declare a constant variable.
Example: 
velt myVariable = 42;
const PI = 3.14;

Importing Modules

To import modules in jsr, use the include keyword. Example: 

#include

Output

To print output to the console, use the print() function. Example: 

print("Hello, world!");

Input

To read input from the user, use the input() function. Example: 

velt name = input("Enter your name:");

Exit

To exit the program, use the exit() function. Example: 

exit();

Clearing the Console

To clear the console, use the clear() function. Example: 

clear();

Type Conversion

To convert values to specific types, use the following functions:

  • num() Converts a value to a number.
  • str() Converts a value to a string.
Example: 
velt number = num("42");
velt string = str(3.14);
